☁️ Cloud Computing Concepts Covered

  • PaaS (Platform as a Service) Deployment: The application is designed to be deployed on managed cloud platforms (like Render), abstracting infrastructure management (VMs) and implicitly utilizing platform load balancing and TLS termination.
  • Asynchronous Processing & Decoupling (Scaling): Network and port scanning workloads are offloaded to background tasks to keep the main HTTP API non-blocking and highly responsive.
  • Environment Awareness & Cloud Security: The application programmatically detects if it is running in a public cloud and gracefully disables restricted operations (like Layer-2 ARP scans) to comply with multi-tenant security rules.
  • Database Abstraction (Data Migration): By using SQLAlchemy ORM, the application’s logic is decoupled from the underlying database engine, enabling seamless migration from local SQLite to a managed cloud database without rewriting queries.
